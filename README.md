# oppProject
Целью данной курсовой работы является разработка программного приложения «MediaLibrary», предназначенного для хранения и систематизации фильмов и сериалов, предоставляющего механизм поиска по названию и отображение списка элементов в удобной форме. Приложение  позволяет  добавлять новые записи, просматривать  сохраненные и искать в каталоге.
